# Rogers Tire & Auto Website

Netlify-ready static site.

## How to upload
1. Unzip this folder.
2. Drag the folder into Netlify deploys, or upload the ZIP directly.
3. Open the website.
4. Click **Admin** in the top navigation.

## Admin features
- Add, edit, and delete services.
- Service changes update the public homepage service cards.
- Create purchase orders from saved services/prices.
- Editable purchase order preview.
- Save draft purchase orders in browser storage.
- Download purchase order as PDF.

## Notes
This version stores admin data in browser localStorage. That works for a starter demo, but for multi-user access you would want Supabase or another database.
